# 🌍 LinguaVerse AI

An AI-powered multilingual translation platform built with the MERN stack, TypeScript, and Python.

LinguaVerse AI aims to provide a unified platform for text, speech, document, and image translation while making communication accessible across multiple languages.

> 🚧 This project is currently under active development.

---

## ✨ Current Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 🏠 Modern Landing Page
- 📊 Responsive Dashboard
- 🧭 Protected Routing
- 🎨 Reusable UI Components
- 📱 Responsive Design
- ⚡ TypeScript Support
- 🔄 REST API Architecture

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/LinguaVerse-AI.git
cd LinguaVerse-AI
```

---

## Frontend

```bash
cd client
npm install
npm run dev
```

Runs on

```
http://localhost:5173
```

---

## Backend

```bash
cd server
npm install
npm run dev
```

Runs on

```
http://localhost:5000
```
